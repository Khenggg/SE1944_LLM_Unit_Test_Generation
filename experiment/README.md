# Experiment Logs

Use this folder to record what happened during AI generation, manual testing, and tool execution.

- `ai_generation_log.csv`: model, prompt version, compile status, and edits for AI-generated tests.
- `manual_test_log.csv`: manual test author, time spent, and fairness checks.
- `tool_run_log.csv`: JaCoCo/PIT commands, exit status, and report paths.
